Key Terminology

The following concepts are fundamental to understanding how HTTP has evolved to support the modern web [cite: 280, 281].

Concept Description
HTTP (Hypertext Transfer Protocol) The communcication system that allows browsers to fetch pages via URLs [cite: 276, 277]
Mulitplexing A feature of HTTP/2 that allows multiple requests to be sent at once over a single connection[cite: 300].
HTTPS HTTP layered on top of TLS encryption to ensure data cannot be read by attackers[cite: 323, 324].
QUIC A UDP-based transport protocol used by HTTP/3 for faster, more secure connections[cite: 306, 307]
Latency The delay in data transfer, which modern protocols like HTTP/3 aim to reduce[cite: 309]
