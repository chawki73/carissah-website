The History of HTTP

HTTP/0.9 and 1.0

HTTP started as a simple way to move research papers. HTTP/0.9 could only fetch a single HTML file with no images[cite: 283, 284]. HTTP/1.0 added headers and support for different content types in 1996[cite: 287].

HTTP/1.1

Introduced in 1997, HTTP/1.1 added persistent connections so browsers didn't have to keep reconnecting for every single file[cite: 292, 293].
