The Early Evolution of HTTP

The Beginnings (HTTP/0.9)

Created in 1989 by Tim Berners-Lee, the first version of HTTP was incredibly simple. It could only fetch a single HTML file and did not support images or stylesheets [cite: 283-284].

Standardization (HTTP/1.0 and 1.1)

HTTP/1.0, standardized in 1996, added headers and support for different content types[cite: 287]. However, it required a new connection for every single request[cite: 289]. HTTP/1.1 improved this in 1997 by introducing persistent connections, allowing the browser to keep a connection open for multiple files [cite: 292-293].
